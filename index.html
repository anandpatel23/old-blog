Projects

MIPS-32 CPU Processor

We developed a complete VHDL description of the MIPS-32 CPU and simulated it by executing a set of programs to demonstrate correct functionality of the processor. The simulation, debugging, and testing of the VHDL code were conducted in ModelSim.

"Let's Go Fishing" - Pokemon Fishing Game

We created a Pokemon based fishing game using a MATLAB GUI as the platform. This was interfaced with an accelerometer that used the arduino as a microcontroller. We were able to successfully implement features such as a Pokemon like loading screen, reeling and casting a fishing rod using the accelerometer and simulating the action and results within MATLAB.

CIGNA "Train YOU" Application

Role: Creator, Project Manager, and User Experience Designer

Within my first co-op at CIGNA, I led the TECDP co-ops to create a proof of concept application called Train YOU as our final project. This consisted of an iOS/Android based application that would encourage Cigna insured users to become healthier and help with the "GO YOU" initiative. Features: -QR Code Scanner of labels on a Gym Equipment Machine with viewable instructions on how to use sed machine with animations and text -Worked with Healthy Life Coaches to create Training Plan portion of application with workout plan instructions and the ability to request a plan -Integration of Healthy Life Coaches: Call a coach, Message a Coach, Incentives, Request a Plan -Biometrics Measurements: Body Mass Index, Fat Loss, Muscle Gain

Bio-sensing Device for Medical Applications

In short, we were able to visualize bio-feedback and response systems to simulate signals visualized in medical applications. This implemented through an Arduino controlled wristband to monitor hypoglycemia through heart rate and perspiration. Check out our blog here.

Autonomous Waste Robot

A quarter into the winter semester during freshman year, I was responsible for building and programming an autonomous waste disposal robot using Lego NXT since my teammates had to leave the class for unknown reasons. This was a very exciting challenge for me since I was able to have experience with Lego NXT in High School. I was introduced to sensor operation, algorithm design and robotics through hands-on-design and course-wide challenge in a decision-making environment. As an objective, I was able to design and build a robot able to autonomously separate trash and nuclear waste, delivering the trash to an incinerator and the nuclear waste to a depot for safe disposal.
